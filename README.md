## рҹ‘Ӣ ApresentaГ§ГЈo

OlГЎ, sou **Acib ABBADE** e desenvolvi o **Dr_Finance** como projeto do Bootcamp DIO - Lab BIA do Futuro.

Escolhi criar um **agente de controle de gastos pessoais** porque identifiquei que 90% das pessoas nГЈo tГӘm controle real dos seus gastos. Recebemos extratos do banco todos os dias, mas nГЈo analisamos padrГөes.

Minha soluГ§ГЈo automatiza essa anГЎlise usando IA local, garantindo privacidade total dos dados.
